# MyBlog

vue+node 的一个全栈项目，已经部署上线。
[小阮的个人博客](http://47.115.83.172/)

### 运用的技术栈

    - vue
    - express
    - mysql
    - nuxt

### 用到的前端 UI 框架

    - 后台管理页面是用的 elementUI
    - 前台展示页面用的是 vuetify

### 功能

    - 后台实现了对数据的增删改查
    - 对数据用validate进行了校验
    - 运用了jwt来进行token验证
    - 实现了分页效果
    - 进行了权限管理
    - 前台实现了响应式布局
    - 实现了留言板功能和留言回复功能
    - 新增文章主题和文章分类
    - 新增学习链接
