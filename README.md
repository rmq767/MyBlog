# MyBlog

vue+node 的一个全栈项目，已经部署上线。
[小阮的个人博客](http://47.115.83.172/)

### 运用的技术栈

- vue cli3
- express
- mysql
- nuxt

### 用到的前端 UI 框架

- 后台管理是用的 elementUI
- 前台展示用的是 vuetify

### 功能

    - 后台实现了对数据的增删改查
    - 对数据用validate进行了校验
    - 运用了jwt来进行token验证
    - 实现了分页效果
    - 进行了权限管理
    - 前台实现了对数据的展示
    - 实现了留言板功能和留言回复功能
